# はじめ方

NoraBar は、起動すると画面上端の中央に小さく常駐します。普段はほとんど見えませんが、マウスを近づけると音楽HUDとして開きます。

## 必要な環境

- Windows 10 以降
- [.NET Desktop Runtime 10](https://dotnet.microsoft.com/download/dotnet/10.0)

NoraBar は `net10.0-windows` を対象にしているため、実行には .NET 10 Desktop Runtime が必要です。利用している Windows のアーキテクチャに合う最新の .NET Desktop Runtime 10.x をインストールしてください。SDK、Visual Studio、.NET CLI は通常利用には不要です。

## 起動する

GitHub Releases から配布ファイルをダウンロードして起動します。

### ポータブル版

1. ポータブル版の zip ファイルをダウンロードします。
2. 任意のフォルダーへ展開します。
3. 展開したフォルダー内の NoraBar 実行ファイルを起動します。

### インストーラー版

1. インストーラーをダウンロードします。
2. インストーラーを実行します。
3. インストール後、NoraBar を起動します。

ソースコードからビルドして起動する手順は [[開発ガイド|開発ガイド]] を参照してください。

## 最初の触り方

1. ソフトウェアでメディアを再生します。
2. 画面上端の中央付近へマウスを移動します。
3. HUD が開き、曲名、アーティスト、アルバムアート、操作ボタンが表示されます。
4. マウスを離すと、HUD は細い待機状態へ戻ります。

## 音楽操作

HUD から次の操作ができます。

- 前の曲へ戻る
- 再生 / 一時停止
- 次の曲へ進む

Windows のグローバルメディアセッションを使うため、対応している音楽アプリやブラウザ再生と連携します。

## 設定を開く

設定画面は次のどちらかから開けます。

- システムトレイの NoraBar アイコンをダブルクリック
- HUD を右クリックして「設定...」を選択

通常起動時は設定画面も自動で開きます。スタートアップ起動時は `--startup` 引数が付き、設定画面を開かずに常駐します。

## 終了する

システムトレイのメニュー、または HUD の右クリックメニューから「終了」を選びます。終了時は HUD が上方向へフェードアウトします。
