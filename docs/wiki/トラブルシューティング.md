# トラブルシューティング

NoraBar が思った通りに表示されないときは、ここから確認してください。

## HUD が表示されない

- 画面上端の中央付近へマウスを移動してください。
- NoraBar は通常時 `200x2` の細い領域として待機します。見た目にはほとんど分からない状態が正常です。
- システムトレイに NoraBar アイコンがあるか確認してください。

## 曲名が表示されない

- 音楽アプリ・ソフトウェア側でメディアを再生してください。

## 再生ボタンが効かない

- 対象の音楽アプリが Windows のグローバルメディア操作に対応している必要があります。
- 複数の音楽アプリを同時に開いている場合、Windows が現在のメディアセッションとして認識しているアプリが操作対象になります。

## 波形が動かない

- PC から音が出ているか確認してください。
- 出力デバイスが有効か確認してください。
- WASAPI ループバックの初期化に失敗した場合、波形は表示されません。アプリを再起動すると復帰することがあります。

## 歌詞が表示されない

- 設定画面で「同期歌詞表示」がオンになっているか確認してください。
- 歌詞は LRCLIB から取得するため、すべての曲で表示できるとは限りません。
- ネットワーク接続を確認してください。
- 曲名、アーティスト名、アルバム名、再生時間を使って検索するため、メディア情報が正しくない場合は一致しないことがあります。
- 曲を切り替えた直後は、歌詞の更新に少し時間がかかる場合があります。連続スキップ時の不要なリクエストを抑えるため、NoraBar は歌詞取得前に短い待機時間を入れています。

## 設定が保存されない

設定ファイルの保存先は次の場所です。

```text
%AppData%\NoraBar\settings.json
```

このフォルダーに書き込み権限があるか確認してください。

## 自動起動しない

設定画面で「システム起動時に実行」をオンにすると、Windows のスタートアップフォルダーに `NoraBar.lnk` が作成されます。

スタートアップフォルダーからショートカットが削除されていないか確認してください。

## 更新確認に失敗する

更新確認は GitHub Releases にアクセスします。ネットワーク接続、プロキシ、GitHub へのアクセス制限を確認してください。
