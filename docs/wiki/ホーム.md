# NoraBar Wiki

ようこそ。ここは、画面上端にそっと住む音楽HUD「NoraBar」のWikiです。

NoraBar は、作業中の視界を散らかさずに、再生中の音楽を少しだけ近くへ置くための Windows デスクトップアプリです。普段は 2px の細いエリアとして待機し、マウスを近づけると曲情報、アルバムアート、再生コントロール、波形ビジュアライザーを備えたHUDとして開きます。

## NoraBar の見どころ

- 画面上端に常駐し、ホバー時だけ開く控えめなHUD
- ドラッグによるウィンドウ位置のカスタム配置機能
- Windows のメディアセッションと連携する音楽コントローラー
- LRCLIB と連携する同期歌詞表示
- WASAPI ループバックで動く 8 バーの波形ビジュアライザー
- Minimal と Productivity の 2 つのデザインスタイル
- 日本語 / English の表示切り替え
- システムトレイ常駐と Windows スタートアップ登録
