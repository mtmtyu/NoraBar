# Security Policy

## Supported Versions

Currently, security updates are provided for the latest version of NoraBar.

| Version | Supported |
| ------- | --------- |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you find a security vulnerability in this project, please do not disclose it publicly by creating a GitHub Issue. Instead, please report it privately through one of the following methods:

- **GitHub Private Vulnerability Reporting**:
  1. Navigate to the main page of the repository on GitHub.
  2. Click the **Security** tab.
  3. Under "Security", click **Advisories** in the left sidebar.
  4. Click **Report a vulnerability**.
  5. Fill out the details of the vulnerability and submit the report.

We appreciate your cooperation in responsibly disclosing security issues to help keep NoraBar secure.
